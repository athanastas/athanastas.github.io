Essential Network Commands

1) ping — Check if a host is alive
ping x.x.x.x     // try: ping 8.8.8.8 or ping www.google.com

2) nslookup — Resolve DNS / find mail server IPs
nslookup www.google.com
// Find mail servers:
nslookup → set type=mx → yahoo.com

3) tracert — Trace packet hops to destination
tracert x.x.x.x    // or tracert www.google.com

4) arp — Show ARP table / detect ARP poisoning
arp -a

5) route — Show routing table, gateway, interface
route print

6) ipconfig — IP, Gateway, DNS info
ipconfig /all
ipconfig /release    // release dynamic IP
ipconfig /renew     // renew IP

7) netstat — Connection stats
netstat -a    // all listening ports with DNS
netstat -n    // all connections with IPs
netstat -an   // combined

net view \\x.x.x.x                     // list shared folders
net use \\ip\ipc$ "" /user:administrator   // connect as admin
net use K: \\computer\C$               // map C drive to K:

Windows Performance & Tweaks

1) Win+R → gpedit.msc → Local Computer Policy → Computer Config → Admin Templates → Windows Components → MS Defender → MAPS → Send files samples (enable: always prompt)

2) Win+R → sysdm.cpl → Advanced → Performance Settings → Adjust for best performance

3) Win+R → msconfig → Boot → Advanced Options → Number of processors: MAX

4) System → Display → Graphics → Default graphics settings → Optimization for windowed games: ON

Shift+Ctrl+Win+B  — Restart video drivers (when frozen)
Ctrl+Shift+T      — Reopen accidentally closed browser tab

Win+R → mrt → Full scan    // Remove viruses

// Slow CPU fix:
CMD (admin): powercfg /setacvalueindex scheme_current sub_processor toldisable 000
CMD (admin): powercfg /setactive scheme_current

// Windows Activation:
PowerShell (admin): irm https://massgrave.dev/get | iex

// Find saved WiFi passwords:
netsh wlan show profiles
netsh wlan show profile name="PROFILE_NAME" key=clear

// Battery report:
CMD admin: powercfg /batteryreport

// Reset BIOS password: bios-pw.org

VPN Protocol Guide

UDP    — Speed (lower latency)
IKEv2 — Mobility + strong security (ideal for mobile)
Xray   — Advanced privacy & anti-censorship
